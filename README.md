# CampNg

## Getting started
Open a terminal and run:
```
git clone https://github.com/gaslight/campng-sf.git
cd campng-sf
open index.html
open jasmine-runner.html
```

## List all labs
```
git tag
```

## Checkout a lab
```
git checkout lab5
```

## Running a server (> lab6)
Make sure to have [node.js](http://nodejs.org/) installed

```
node server.js
```
